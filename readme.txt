Disables successful auto-update emails for WordPress core, plugins and themes.

== Description ==

WordPress sends email notifications for auto-updates by default, including when updates succeed and when they fail. In many cases, successful update emails can clutter your inbox, while failure emails remain critical.

Stupid Simple Update Email Disabler:

Suppresses all success auto-update emails for core, plugins, and themes.

Continues to send fail and upgrade notifications so you’re informed of issues and major core upgrades.

No settings screen—just install and activate.

== Frequently Asked Questions ==

= Will I still get notified of failed updates? =

Yes—only successful update emails are suppressed. Failure and upgrade notices still send.
